## Install

Make sure to clone into `$HOME/`

```sh
git clone git@github.com:martendebruijn/dotfiles.git
cd dotfiles
./install.sh
```

Enter environment variables in `.zshenv`.

You may need to give the script execute permissions:

```sh
sudo chmod +x install.sh
```
